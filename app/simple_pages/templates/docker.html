Introduction
What is Docker?

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers. The software that hosts the containers is called Docker Engine.

What is Docker hub ?

Docker Hub is a service provided by Docker for finding and sharing container images with your team.

Docker commands

docker ps: check running containers
docker conatins ls: Check docker container present in system
docker images ls: Check docker images present in system
docker kill : Kill container
docker stop : Stop container

Access container using terminal

  • Use docker ps to get the name of the existing container
  • Use the command ocker exec -it /bin/bash to get a bash shell in the container
  • Generically, use docker exec -it to execute whatever command you specify in the container.

Run project

  • To Build with docker compose: docker compose up --build
  • To run tests, Lint, and Coverage report use this command: pytest --pylint --cov
